Mastodon Network Monitoring Project

The Mastodon Network Monitoring Project is a statistic graphing website created by @eliotberriot@mastodon.eliotberriot.com. It displays instances on a global map, along with other gathered data, in real time. Some other stats collected are: instance, status, and user counts, and hashtag trends.
