Write Freely(ライト フリーリー)とは、オープンソースの分散SNSプラットフォーム。ブログサービスを提供する。

概要[編集 | ソースを編集]

Write FreelyはWrite.asの開発者Matt Baer氏によって開発されたブログプラットフォームでWrite.asと同様のブログの公開・閲覧ができる機能を備えているが、Write.asと異なりすべての機能が自由に利用でき、WordPressのようなセルフホストを前提としたオープンソースソフトウェアである。
