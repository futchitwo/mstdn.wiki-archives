Toot-counter

toot-counter
開発者 @MulticolorWorld@mstdn.maud.io
ソースコード プロプライエタリ
プラットホーム Web
サービス開始日 2017年12月18日[1]
Webサイト https://toot-counter.net


toot-counterは、マストドンの一日のトゥート数を自動でトゥートするウェブアプリケーション。開発者は、Twitter向けアプリケーション「ツイ廃あらーと」の作者でもある@MulticolorWorld@mstdn.maud.io氏。

登録すると、JST（日本標準時）0時に自動で1日のトゥート数をトゥートする。
